# 前端连接后端服务指南

> 本文档说明 React 前端如何连接后端服务

---

## 一、后端服务信息

| 项目 | 值 |
|------|-----|
| 后端 IP 地址 | `192.168.21.17` |
| 后端端口 | `8000` |
| API 基础路径 | `/api/v1` |
| 完整 API 地址 | `http://192.168.21.17:8000/api/v1` |
| WebSocket 地址 | `ws://192.168.21.17:8000/ws` |

## 二、环境要求

- 前端电脑与后端电脑需要在**同一个局域网**（同一个 WiFi）
- 后端服务已启动并正常运行
- 防火墙允许 8000 端口访问

## 三、验证后端服务

在浏览器中访问以下地址，验证后端是否正常运行：

```
http://192.168.21.17:8000/api/v1/health
```

## 四、React 项目配置

### 4.1 创建环境变量文件

在 React 项目根目录创建 `.env` 文件：

```
REACT_APP_API_BASE_URL=http://192.168.21.17:8000/api/v1
REACT_APP_WS_URL=ws://192.168.21.17:8000/ws
```

如果使用 Vite 创建的 React 项目，创建 `.env` 文件：

```
VITE_API_BASE_URL=http://192.168.21.17:8000/api/v1
VITE_WS_URL=ws://192.168.21.17:8000/ws
```

### 4.2 创建 API 配置文件

创建 `src/config/api.js`：

```javascript
const API_BASE_URL = import.meta.env.VITE_API_BASE_URL || "http://192.168.21.17:8000/api/v1";
const WS_URL = import.meta.env.VITE_WS_URL || "ws://192.168.21.17:8000/ws";

export { API_BASE_URL, WS_URL };
```

## 五、API 请求示例

### 5.1 使用 fetch 发送请求

```javascript
import { API_BASE_URL } from "./config/api";

async function getGameList() {
  const response = await fetch(`${API_BASE_URL}/game/list`);
  return response.json();
}

async function createRoom(data) {
  const response = await fetch(`${API_BASE_URL}/room/create`, {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
    },
    body: JSON.stringify(data),
  });
  return response.json();
}

async function joinRoom(roomCode, data) {
  const response = await fetch(`${API_BASE_URL}/room/join`, {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
    },
    body: JSON.stringify({ room_code: roomCode, ...data }),
  });
  return response.json();
}
```

### 5.2 使用 axios 发送请求

首先安装 axios：

```bash
npm install axios
```

创建 `src/api/index.js`：

```javascript
import axios from "axios";
import { API_BASE_URL } from "../config/api";

const api = axios.create({
  baseURL: API_BASE_URL,
  headers: {
    "Content-Type": "application/json",
  },
});

export default api;
```

使用示例：

```javascript
import api from "./api";

async function getGameList() {
  const response = await api.get("/game/list");
  return response.data;
}

async function createRoom(data) {
  const response = await api.post("/room/create", data);
  return response.data;
}

async function joinRoom(roomCode, data) {
  const response = await api.post("/room/join", { room_code: roomCode, ...data });
  return response.data;
}
```

### 5.3 带认证的请求

如果需要携带 Token：

```javascript
import api from "./api";

api.interceptors.request.use((config) => {
  const token = localStorage.getItem("token");
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});
```

## 六、WebSocket 连接示例

### 6.1 基础 WebSocket 连接

```javascript
import { WS_URL } from "./config/api";

let socket = null;

function connectWebSocket() {
  socket = new WebSocket(WS_URL);

  socket.onopen = () => {
    console.log("WebSocket 已连接");
  };

  socket.onmessage = (event) => {
    const data = JSON.parse(event.data);
    console.log("收到消息:", data);
  };

  socket.onerror = (error) => {
    console.error("WebSocket 错误:", error);
  };

  socket.onclose = () => {
    console.log("WebSocket 已断开");
  };
}

function sendMessage(message) {
  if (socket && socket.readyState === WebSocket.OPEN) {
    socket.send(JSON.stringify(message));
  }
}

function disconnectWebSocket() {
  if (socket) {
    socket.close();
  }
}

export { connectWebSocket, sendMessage, disconnectWebSocket };
```

### 6.2 React Hook 封装

创建 `src/hooks/useWebSocket.js`：

```javascript
import { useState, useEffect, useRef, useCallback } from "react";
import { WS_URL } from "../config/api";

function useWebSocket() {
  const [isConnected, setIsConnected] = useState(false);
  const [messages, setMessages] = useState([]);
  const socketRef = useRef(null);

  const connect = useCallback(() => {
    socketRef.current = new WebSocket(WS_URL);

    socketRef.current.onopen = () => {
      setIsConnected(true);
      console.log("WebSocket 已连接");
    };

    socketRef.current.onmessage = (event) => {
      const data = JSON.parse(event.data);
      setMessages((prev) => [...prev, data]);
    };

    socketRef.current.onclose = () => {
      setIsConnected(false);
      console.log("WebSocket 已断开");
    };

    socketRef.current.onerror = (error) => {
      console.error("WebSocket 错误:", error);
    };
  }, []);

  const disconnect = useCallback(() => {
    if (socketRef.current) {
      socketRef.current.close();
    }
  }, []);

  const sendMessage = useCallback((message) => {
    if (socketRef.current && socketRef.current.readyState === WebSocket.OPEN) {
      socketRef.current.send(JSON.stringify(message));
    }
  }, []);

  useEffect(() => {
    return () => {
      disconnect();
    };
  }, [disconnect]);

  return { isConnected, messages, connect, disconnect, sendMessage };
}

export default useWebSocket;
```

使用示例：

```jsx
import useWebSocket from "./hooks/useWebSocket";

function GameRoom() {
  const { isConnected, messages, connect, sendMessage } = useWebSocket();

  useEffect(() => {
    connect();
  }, [connect]);

  const handleRollDice = () => {
    sendMessage({ type: "roll_dice", gameId: "123" });
  };

  return (
    <div>
      <p>连接状态: {isConnected ? "已连接" : "未连接"}</p>
      <button onClick={handleRollDice}>掷骰子</button>
      <ul>
        {messages.map((msg, index) => (
          <li key={index}>{JSON.stringify(msg)}</li>
        ))}
      </ul>
    </div>
  );
}
```

## 八、常见问题

### 8.1 无法连接后端

**检查项：**

1. 确认后端服务已启动
2. 确认前后端在同一局域网
3. 检查防火墙是否允许 8000 端口
4. 尝试在浏览器直接访问 `http://192.168.21.17:8000/api/v1/health`

### 8.2 跨域错误

如果浏览器控制台出现 CORS 错误：

```
Access to XMLHttpRequest at 'http://192.168.21.17:8000' from origin 'http://localhost:3000' 
has been blocked by CORS policy
```

**解决方案：** 后端已配置 CORS，允许所有来源访问。如果仍有问题，请检查后端是否正常运行。

### 8.3 WebSocket 连接失败

**检查项：**

1. 确认 WebSocket 地址正确：`ws://192.168.21.17:8000/ws`
2. 检查后端 WebSocket 服务是否正常
3. 查看浏览器控制台错误信息

## 九、测试连接

### 9.1 测试 API 连接

在 React 组件中测试：

```jsx
import { useEffect } from "react";
import { API_BASE_URL } from "./config/api";

function App() {
  useEffect(() => {
    fetch(`${API_BASE_URL}/health`)
      .then((res) => res.json())
      .then((data) => console.log("后端连接成功:", data))
      .catch((err) => console.error("后端连接失败:", err));
  }, []);

  return <div>测试连接</div>;
}

export default App;
```

### 9.2 测试 WebSocket 连接

```jsx
import { useEffect } from "react";
import { WS_URL } from "./config/api";

function App() {
```
